# 🚀 ANARTECH Balta V1.0

## 📌 Descripción
Balta es un cachorro virtual interactivo con modos especiales: **Modo VR** y **Modo Arquitecto**.  
Este proyecto combina HTML, CSS y JavaScript para crear una experiencia visual atractiva y educativa.

## 🎯 Objetivo
Desarrollar una página web interactiva que permita jugar con Balta y activar efectos especiales como **Realidad Virtual** y **Arquitectura**.

## 📂 Estructura del Proyecto
- `index.html` → Interfaz principal.
- `style.css` → Estilos y animaciones.
- `script.js` → Funcionalidad e interacciones.
- `pug_black.png` → Imagen base de Balta.
- `balta_arquitecto.png` → Imagen de Balta como arquitecto.

## 🛠️ Instalación y Uso
1. Clona el repositorio:  

git clone https://github.com/tu-repo/balta-puppy.git

2. Abre `index.html` en un navegador.
3. Interactúa con Balta mediante los botones.
