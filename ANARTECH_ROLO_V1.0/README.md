# Estructura del proyecto /rolo-the-puppy
Este proyecto  se realiza para crear una página web interactiva que presenta a "Rolo", un cachorro (o "pug") que tiene animaciones, diseño futurista y diferentes imágenes.
 A continuación, te explico cada archivo y su función:

* 1. index.html (Interfaz principal)
Es el archivo principal de la página web.
Contiene el HTML, que es la estructura base del contenido que se mostrará.
Aquí se define el esqueleto de la página, como los encabezados, textos, botones, imágenes y cualquier otro elemento visible en el navegador.
También puede contener referencias a otros archivos como style.css (para el diseño) y script.js (para la lógica interactiva).
* 2. style.css (Diseño y animaciones futuristas)
Este archivo contiene las reglas de CSS (Hojas de Estilo en Cascada) que definen la apariencia visual de la página.
Se utiliza para personalizar colores, tipografías, tamaños, posiciones y estilos generales de los elementos del HTML.
La mención de "animaciones futuristas" sugiere que este archivo también incluye animaciones avanzadas, como efectos de transición, transformaciones 3D o movimientos dinámicos para darle un aspecto moderno y atractivo.
* 3. script.js (Interacciones y lógica de animación)
Este archivo contiene código en JavaScript, que se encarga de añadir interactividad y lógica a la página.
Puede incluir funciones para manejar eventos como clics en botones, movimientos del ratón, o teclas presionadas.
También puede controlar las animaciones mencionadas en el archivo style.css, haciendo que respondan a las acciones del usuario.
Es probable que este archivo sea responsable de interacciones específicas relacionadas con "Rolo", como mostrar diferentes imágenes o activar animaciones cuando el usuario interactúe con la página.
* 4. pug_black.png (Imagen de Rolo)
Es una imagen estática que muestra a "Rolo", el cachorro pug.
Esta imagen podría ser utilizada como representación inicial o principal del personaje en la interfaz.
* 5. pug_vr.png (Imagen de Rolo con gafas VR)
Es otra imagen estática que muestra a "Rolo" con gafas de realidad virtual (VR).
Podría ser utilizada para representar un estado o interacción específica en la página, como cuando el usuario activa una función relacionada con tecnología o realidad virtual.
