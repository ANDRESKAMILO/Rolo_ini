# 🐾 Proyecto Rolo_ini

Este repositorio contiene dos versiones de un proyecto educativo e interactivo centrado en cachorros virtuales: **Rolo** y **Balta**. Ambas versiones exploran conceptos de desarrollo web, animaciones y lógica interactiva usando HTML, CSS y JavaScript.

## 📦 Versiones Disponibles

- **[ANARTECH_ROLO_V1.0](https://github.com/ANDRESKAMILO/Rolo_ini/tree/rolo-v1)**  
  Página web interactiva protagonizada por "Rolo", un cachorro pug con animaciones futuristas y diferentes imágenes.  
  Incluye explicación detallada de la estructura y archivos del proyecto.

- **[ANARTECH_ROLO_V2.0](https://github.com/ANDRESKAMILO/Rolo_ini/tree/balta-v1)**  
  Proyecto interactivo con "Balta", un cachorro virtual que cuenta con modos especiales: **Modo VR** y **Modo Arquitecto**.  
  Permite experimentar con efectos visuales y educativos.

## 🚀 ¿Cómo usar este repositorio?

1. **Elige la versión que deseas explorar:**
   - [Rolo-v1](https://github.com/ANDRESKAMILO/Rolo_ini/tree/rolo-v1): Rolo, el pug animado.
   - [Balta-v1](https://github.com/ANDRESKAMILO/Rolo_ini/tree/balta-v1): Balta, con modos VR y Arquitecto.

2. **Clona el repositorio y cambia a la rama deseada:**
   ```sh
   git clone https://github.com/ANDRESKAMILO/Rolo_ini.git
   cd Rolo_ini
   git checkout rolo-v1   # o balta-v1
   ```

3. Abre index.html en tu navegador para comenzar a interactuar con el cachorro virtual.
